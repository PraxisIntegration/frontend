Customer Form - Iframe Integration Example

This example demonstrates how to embed the React customer form as an iframe with query parameters.

Query parameters passed: currencyFrom=USD¤cyTo=EUR&id=12345&amount=1000&theme=light

Try both themes:

Customer Information Form

Integration Notes:

  • The iframe automatically extracts and displays query parameters
  • Form is fully responsive for desktop, tablet, and mobile devices
  • All form fields include comprehensive validation
  • Date field enforces MM/DD/YYYY format and validates date range (1900 - today)
  • Phone field accepts international format with country codes
  • Country dropdown includes all ISO 3166-1 alpha-2 country codes
  • Form submits to a mock endpoint (JSONPlaceholder) for testing

Query Parameters:

  • currencyFrom - Source currency code
  • currencyTo - Target currency code
  • id - Transaction or reference ID
  • amount - Transaction amount

Example iframe code:

<iframe src="http://localhost:5173/?currencyFrom=USD¤cyTo=EUR&id=12345&amount=1000" width="100%" height="800" frameborder="0"></iframe>
